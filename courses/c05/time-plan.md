# C05 GitHub Copilot基礎 -- 時間配分設計

合計: 8時間(480分)

## コースの特徴
GitHub Copilotの基本操作と日常開発への組み込み方。心理的ハードルを下げる設計。
法人研修で実証した「因果関係体験型」アプローチ -- 「こうやると、こう返ってくる」を体で覚える。

## セクション別時間配分

| Sec | タイトル | 講義 | ハンズオン | 小計 |
|-----|---------|------|----------|------|
| 01 | GitHub Copilotとは -- 仕組み、料金、プラン比較 | 30min | 15min(セットアップ+初めてのサジェスト) | 45min |
| 02 | コード補完 -- インラインサジェストを使いこなす | 20min | 30min(Tab受入れ、候補切替、拒否判断を体感) | 50min |
| 03 | Copilot Chat -- 対話でコードを理解・改善する | 20min | 30min(質問、説明、リファクタ、エラー解析) | 50min |
| -- | 復習A(Sec01-03) | -- | 25min(補完+Chatを組合せた小機能実装) | 25min |
| 04 | インラインチャット -- コードの中で指示を出す | 20min | 30min(選択範囲指示、部分修正、コメント生成) | 50min |
| 05 | コメントドリブン開発 -- コメントを先に書く型 | 20min | 30min(関数3つをコメントから生成) | 50min |
| -- | 復習B(Sec04-05) | -- | 25min(コメント設計→インラインで実装の一連フロー) | 25min |
| 06 | テスト生成とリファクタリング | 20min | 30min(既存コードのテスト自動生成+リファクタ) | 50min |
| 07 | セキュリティとガバナンス -- ライセンス問題、社内ポリシー | 25min | 15min(Copilot Business管理画面+ポリシー設定) | 40min |
| 08 | トレンド -- Agent Mode, Workspace, Multi-file editing | 20min | -- | 20min |
| 09 | 総合ハンズオン: CLIツールをCopilotで完成させる | 10min | 65min | 75min |
| -- | 合計 | 185min | 295min | 480min |

ハンズオン比率: 61%

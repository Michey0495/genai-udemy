# C04 Claude Code基礎 ターミナルAI開発入門 -- 時間配分設計

合計: 8時間(480分)

## コースの特徴
Claude Code CLIツールの基本操作からプロジェクト設定まで。
ターミナルベースのAI開発が何を変えるかを体験する入門コース。
公式ドキュメントが英語のみのため、日本語で体系的に学べる唯一のコース。

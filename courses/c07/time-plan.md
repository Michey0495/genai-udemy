# C07 Claude Code応用 プロダクション開発 -- 時間配分設計

合計: 12時間(720分)

## コースの特徴
Claude Codeを使った本格的なソフトウェア開発。C04基礎の上に、スキル開発、フック、
サブエージェント、仕様駆動開発、チーム運用、CI/CD統合まで踏み込む。
日本語でスキル/フック/エージェント開発を教えるコースは存在しない。
